# ShortestPath

## Dijkstra

* 한 정점에서 다른 모든 정점까지의 최단 거리를 구할 때 이용 
* 음수 그래프에는 사용 불가 

### 동작

* 시작 거리를 0. 다른 정점과의 거리는 INF
* 우선순위 큐에서 가장 거리가 짧은 정점을 꺼낸다
* 해당 정점에서 다른 이웃 정점으로 가는 모든 비용을 갱신한다
* 모든 정점에 대해서 처리한다

## Bellmanford

음수 가중치가 있는 그래프에서도 동작할 수 있고
음수 사이클에 대해서는 최단 거리 계산이 불가능하다. 

### 동작 

* 노드의 개수를 `V` 라고 했을 때 모든 경로를 `V-1` 번 반복해서 갱신한다. 

* `V`번의 갱신에서 값이 변경이 됐다면, 음수 사이클이 있는 상황이다.

## Floyd Warshall 

모든 시작점에서 다른 모든 노드의 정점까지의 

최단 거리를 구하는 알고리즘이다. 

`i에서 j로 갈 때 k를 거치는 것이 더 빠른지 확인한다` 

```swift
for k in 1...N:
  for i in 1...N:
    for j in 1...N:
      dist[i][j] = min(dist[i][j], dist[i][k] + dist[k][j])
```

시간 복잡도는 V^3
