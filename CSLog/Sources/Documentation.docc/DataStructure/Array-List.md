## 배열과 연결 리스트(Linked List)의 차이

> Keyword: 조회, 삽입, 메모리

배열은 메모리 상에 연속적인 공간에 할당된다. 
연결 리스트는 연속적인 공간에 할당되지 않고 노드 안에 다음 노드의 메모리 주소를 저장한다. 

배열은 임의의 원소 접근에 O(1)만큼 사용하고, 연결 리스트는 순차적으로 접근해야하기 때문에 O(n)이 필요하다. 

배열은 중간에 삽입 시 뒤에 있는 원소들의 위치를 바꿔야하기 때문에 O(n)이 필요하고, 연결 리스트는 O(1)이 필요하다.
