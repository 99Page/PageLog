# Set-Dictionary

배열과 리스트에 대해서 정리한 문서

## Set 

* 해시 테이블을 기반으로 동작한다. 삽입, 삭제, 조회에 O(1) 필요. 

* 동일한 값을 중복 저장하지 않는다. 

* 순서를 보장하지 않는다. 

* 집합 연산에 유리하다. (union, substraction 등)

## Dictionary

* key와 value를 대응시켜서 저장한다. 

* 하나의 키는 하나의 value를 갖는다. 

* 해시 테이블을 기반으로 동작한다. 삽입, 삭제, 조회에 O(1) 필요.  

* 순서를 보장하지 않는다.
